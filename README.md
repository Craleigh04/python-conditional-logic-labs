# Python Conditional Logic Labs

This project demonstrates beginner Python conditional logic through a set of small programs focused on decision-making, branching, and user input handling. Each lab uses if, elif, and else statements to solve a different problem.

## Overview
These exercises were completed as foundational Python practice and show how conditional logic can be used to compare values, evaluate multiple outcomes, and control program flow.

## Labs Included
- Favorite word checker using if/elif/else
- Income tax calculator with multiple conditional branches
- Leap year checker using Gregorian calendar rules

## Skills Demonstrated
- Python programming
- Conditional logic
- if / elif / else statements
- User input handling
- Basic debugging and program flow control

- `module3_lab1.py`
- `module3_lab2.py`
- `module3_lab3.py`

## Why This Project Matters
This project reflects early Python problem-solving skills and demonstrates a solid understanding of branching logic, which is essential for more advanced programming, scripting, and data workflows.
